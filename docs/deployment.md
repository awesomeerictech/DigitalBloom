# Deployment Options

DigitalBloom supports multiple deployment models depending on business needs.

---

## Local Device Deployment

- Run directly on Android, Windows, or embedded hardware
- Ideal for small shops, caf√©s, and events
- No internet required for customer access

---

## Embedded / Edge Deployment

- Deploy on Raspberry Pi or similar devices
- Serve content over local Wi-Fi
- Highly reliable for exhibitions and pop-up events

---

## Cloud / VPS Deployment

- Host on a cloud VPS
- Public access via custom domain
- Suitable for multi-branch or distributed businesses

---

## Hybrid Deployment

- Combine local edge devices with cloud management
- Optimised for resilience and scalability

---

Commercial guidance is available for complex deployments.
