# Quick Start Guide

This guide helps you run DigitalBloom quickly using the provided binaries.

---

## Step 1: Download the Binary

Download the appropriate binary from:
- GitHub Releases, or
- The `/binaries/` directory in this repository

Verify the checksum using the provided `checksums.txt`.

---

## Step 2: Install & Launch

### Android
- Install the APK on your device
- Grant required permissions when prompted
- Launch DigitalBloom from the app list

### Windows
- Download the `.exe`
- Run the executable (administrator rights may be required on first launch)
- Allow firewall access if prompted

---

## Step 3: Access the Marketing Interface

Once running, DigitalBloom exposes a local or hosted marketing interface.

- Open the provided URL in a browser
- Use the generated QR code to access the site from other devices

No app installation is required for end users.

---

## Step 4: Add Products or Services

- Upload content using metadata archives (MAT)
- Add images, descriptions, and pricing
- Publish updates instantly

---

## Step 5: Share Your QR Code

- Print QR codes for tables, shelves, counters, or posters
- Use short marketing links for online sharing
- Customers scan and access your marketing page immediately

---

## Offline & Local Use

DigitalBloom can run:
- Fully offline on local Wi-Fi
- On a hotspot created by the host device
- In low-connectivity environments
